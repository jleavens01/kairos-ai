# 📚 세모지(세상의모든지식) 콘텐츠 스타일 가이드

## 🎯 채널 정체성
- **슬로건**: "세상의 모든 지식을 쉽고 재미있게"
- **타겟**: 지적 호기심이 많은 20-40대
- **톤앤매너**: 친근하지만 전문적, 재미있지만 깊이 있는

## ✍️ 세모지 원고 스타일 특징

### 1. 도입부 (Hook)
```
특징적인 패턴:
- "여러분은 혹시 [질문/상황] 알고 계신가요?"
- "오늘은 정말 놀라운 이야기를 들려드리려고 합니다"
- 충격적인 팩트나 통계로 시작
- 일상적 경험과 연결된 도입
```

### 2. 스토리텔링 구조
```
세모지 특유의 3막 구조:
1막 (30%): 호기심 유발 + 배경 설명
2막 (50%): 본격적인 지식 전달 + 흥미로운 에피소드
3막 (20%): 인사이트 + 현재/미래 연결 + 마무리
```

### 3. 언어적 특징
- **경어체 사용**: "~입니다", "~인데요", "~죠"
- **청자 포함 표현**: "우리가", "여러분도", "함께"
- **감탄사 활용**: "놀랍게도", "흥미롭게도", "실제로"
- **비유와 은유**: 복잡한 개념을 일상적 비유로 설명

### 4. 정보 전달 방식
```javascript
{
  "깊이": "전문적 내용을 쉽게 풀어서",
  "속도": "천천히 음미하듯 설명",
  "반복": "핵심 메시지 2-3회 다른 방식으로 반복",
  "예시": "구체적 사례와 데이터 제시"
}
```

### 5. 감정적 연결
- 공감대 형성: "여러분도 한 번쯤은..."
- 놀라움 표현: "믿기 어려우시겠지만..."
- 호기심 자극: "그런데 여기서 끝이 아닙니다"

## 📝 세모지 스크립트 예시

### 브랜드 스토리 (예: 나이키)
```
[도입부]
"Just Do It. 이 짧은 세 단어가 어떻게 전 세계 수십억 명의 
마음을 움직이게 되었을까요? 오늘은 운동화 한 켤레에서 시작해 
인류의 잠재력을 믿는 철학이 된, 나이키의 놀라운 이야기를 
들려드리려고 합니다."

[전개부]
"1964년, 스탠포드 대학의 육상 선수였던 필 나이트는 
일본으로 향하는 비행기에 올랐습니다. 그의 가방 안에는 
단돈 50달러와 엉성한 사업 계획서 한 장이 전부였죠. 
그런데 놀랍게도, 바로 이 순간이..."

[특징적 전환]
"그런데 여기서 정말 흥미로운 점은..."
"하지만 이게 끝이 아닙니다..."
"더 놀라운 사실은..."
```

## 🎨 시각적 스타일
- **썸네일**: 강렬한 대비, 큰 텍스트, 호기심 자극
- **인포그래픽**: 깔끔하고 직관적, 세모지 브랜드 컬러
- **자막**: 핵심 키워드 강조, 적절한 이모지 활용

## 🎙️ 나레이션 스타일
- **속도**: 분당 150-160단어 (느리고 명확하게)
- **톤**: 따뜻하고 친근한, 마치 이야기를 들려주는 듯한
- **강조**: 핵심 단어에서 약간의 pause와 강세
- **감정**: 내용에 따라 놀라움, 감탄, 의문 등 변화

## 📊 콘텐츠 선정 기준
1. **지적 호기심**: 새로운 지식이나 관점 제공
2. **스토리텔링**: 이야기로 풀어낼 수 있는 주제
3. **시의성**: 현재 관심사와 연결 가능
4. **깊이**: 표면적 정보가 아닌 심층 분석
5. **실용성**: 삶에 적용 가능한 인사이트

## 🔄 반복되는 표현 패턴
```
오프닝:
- "안녕하세요, 세상의 모든 지식, 세모지입니다"

전환구:
- "그런데 말입니다"
- "여기서 잠깐"
- "흥미로운 점은"
- "놀랍게도"

클로징:
- "오늘도 새로운 지식과 함께 하셨길 바랍니다"
- "다음에도 더 흥미로운 이야기로 찾아뵙겠습니다"
```

## 💡 핵심 차별화 포인트
1. **깊이 있는 리서치**: 단순 정보 나열이 아닌 통찰
2. **스토리텔링**: 지루하지 않은 서사 구조
3. **감정적 연결**: 청자와의 공감대 형성
4. **시각적 보조**: 이해를 돕는 적절한 비주얼
5. **교육적 가치**: 보고 나면 뭔가 배운 느낌
